## Project Description
Movie App is designed to showcase **API integration and React functionality**. It fetches movie data from a public API (e.g., OMDb API, TMDb API) to display the latest and trending movies.  

Key highlights of the project include:  
- Fetching and displaying real-time movie data from an API  
- Implementing search functionality with **debounce** for improved performance  
- Dynamic rendering of movie cards and details  
- Responsive design for both desktop and mobile devices  

---

## Features

- **Latest Movies:** Browse a curated list of trending movies fetched from a public API.  
- **Search with Debounce:** Search for movies with optimized API calls to reduce unnecessary requests.  
- **Movie Details:** View basic information like title, release date, and ratings.  
- **Responsive Design:** Works on desktops, tablets, and mobile screens.  
- **API Integration:** Demonstrates real-time data fetching and dynamic rendering.
